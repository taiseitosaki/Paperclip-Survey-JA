# Overview figure planning prompt

Distill the survey manuscript into a figure spec before any drawing step.

Create `figures/overview_figure_spec.md` with the following structure:

1. Figure claim (one sentence)
2. Reader takeaway (one sentence)
3. Chosen layout archetype
4. Dominant panel
5. Secondary panels
6. Node list
7. Edge list
8. Representative examples per node (max 3)
9. Content omitted from the figure
10. Caption draft

Hard limits:

- <= 8 major nodes
- <= 12 arrows
- <= 35 words inside any node
- <= 3 examples per node
- no long arrow labels
- no paper-title walls
- no design that simply mirrors the manuscript section outline
