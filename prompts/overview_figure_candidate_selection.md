# Overview figure candidate selection prompt

You have produced at least two overview-figure candidates. Select the best one.

Evaluate each candidate on:

- clarity of the main message;
- hierarchy and panel balance;
- readability at manuscript scale;
- information density;
- faithfulness to the manuscript;
- whether it looks like a journal figure rather than a generic infographic.

Write `reports/figure_selection.md` with:

- short description of each candidate;
- strengths;
- weaknesses;
- selected candidate;
- exact reason for selection;
- any final revision applied before acceptance.
