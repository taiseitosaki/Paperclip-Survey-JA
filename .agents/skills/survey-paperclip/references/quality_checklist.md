# Survey quality checklist

A run is not complete until all items are addressed.

## Coverage

- [ ] At least two search rounds were performed.
- [ ] Query expansion used discovered terms, not only the original prompt.
- [ ] Landmark, historical, recent, benchmark, evaluation, and critique papers were considered.
- [ ] Preprints were retained if relevant and their status was recorded.
- [ ] Evidence card count is proportionate to candidate count or the exception is justified.

## Evidence

- [ ] Every cited paper has an evidence card.
- [ ] Every central technical paper has method/architecture, data/task, objective, evaluation, novelty, and limitation fields.
- [ ] Map failures were recovered or explicitly documented.
- [ ] Abstract-only evidence is labeled.

## Draft

- [ ] English survey was written first.
- [ ] Japanese version was translated from the audited English version.
- [ ] Claims distinguish settled findings from emerging claims.
- [ ] Technical explanations are not just one-line paper summaries.
- [ ] Important disagreements and limitations are included.

## References and export

- [ ] Markdown citations link to reference anchors.
- [ ] English and Japanese TeX files exist.
- [ ] BibTeX file contains all cited IDs.
- [ ] Validation report was generated and reviewed.
