# Overview figure checklist

- Clear academic purpose.
- Faithful to manuscript content.
- Readable labels and logical flow.
- No unnecessary decorative elements.
- Caption-ready and easy to reference in prose.
- Useful in both Markdown preview and TeX output.
- One dominant organizing idea.
- One dominant panel with clear visual hierarchy.
- No dense wall-of-bullets boxes.
- No more than 3 representative examples in any node.
- No long arrow labels.
- Better than a section-outline diagram.
- Rendering route recorded as `image_gen` or documented fallback.
- If image_gen succeeded, no deterministic/code-based renderer was run.
- Accepted image is inserted into Markdown and carried through to TeX.
- Candidate selection documented in `reports/figure_selection.md`.
